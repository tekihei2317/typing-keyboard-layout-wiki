# 月見草エキス配列

[月見草配列](./月見草配列.md)を改良した配列。ゃゅょぃぇの5キーをシフトに使用する。

月見草配列からは、外来音の入力方法と、句読点がシフトキーを兼ねていないところが変わっている。

Alternative Typing Contest 2025で優勝した実績がある。

## 配列図

濁音は分置。清音と拗音、外来音は同置。

外来音は、「ゃぃゅぇょ」を「ぁぃぅぇぉ」として使うようになっている。例えば「ふ + ゃ = ふぁ」。

```
単打
ー か ゃ く ま ゔ が ぃ ぇ む ふ
し ょ う ゅ 、 て た い ん に こ
き で は の け な と す 。 っ 　

ゃ後置シフト
ぢゃ る ＿ ぶ ぴゃ ゔぁ つぁ ＿ ＿ ぎゃ ふぁ
しゃ ＿ ＿ ＿ ＿ せ りゃ ぱ や にゃ ちゃ
きゃ ゎ ＿ ぐ ぁ じゃ さ びゃ ひゃ みゃ

ぃ後置シフト
ぢ ご ＿ っ ぽ ぴ ゔぃ つぃ ＿ ＿ ぞ ふぃ
ね ＿ うぃ ＿ ぺ てぃ り ＿ ゆ げ ち
づ でぃ ＿ ど ぃ じ そ ＿ ひ み

ゅ後置シフト
ぢゅ ＿ ＿ ぴゅ ゔゅ つ ＿ ＿ ぎゅ ふゅ
しゅ ＿ お ＿ ＿ てゅ りゅ ず よ にゅ ちゅ
きゅ でゅ ざ どぅ ぅ じゅ とぅ びゅ ひゅ みゅ

ぇ後置シフト
＿ め ＿ ぬ ＿ ゔぇ つぇ ＿ ＿ ぎ ふぇ
しぇ ＿ うぇ ＿ わ ぼ れ いぇ ＿ ら ちぇ
＿ ＿ へ あ ぇ じぇ ろ び ＿ ＿

ょ後置シフト
ぢょ え ＿ べ ぴょ ゔぉ つぉ ＿ ＿ ぎょ ふぉ
しょ ＿ うぉ ＿ ぜ だ りょ ほ も にょ ちょ
きょ ＿ ぷ ば ぉ じょ を びょ ひょ みょ
```
