# 月林檎配列

めんめんつさん作の、後置シフトの月系配列。キー範囲は33キーで、月配列伝統の32キーに加えてCAPSLOCK位置のキーも使用する。

薙刀式や月見草配列に影響を受けている。

### 説明

清音46文字＋小書10文字（ぁぃぅぇぉゃゅょっゎ）＋記号4文字（ー、。濁音）＝60文字を配置。

清濁同置。一つの位置濁音になるキーは一つしか含まれていないので、シフト面の濁音は2打で入力できる。

一点例外として、ぱ行はま行＋濁点で入力する。

### メモ

ょ後置シフト、濁音の排他的配置による2打鍵化など、アイデアが輝いている。濁音が同置のため記憶負荷が少ないのもいいところ。

CapsLock位置のキーと使用しているのが惜しい。
