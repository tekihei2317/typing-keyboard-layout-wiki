# 月見草配列

順次打鍵の計算配列。1800万文字のデータを集め、打鍵のつながりが滑らかになるように設計されている。

ゃゅょを後置シフト、っを前置シフトと兼用していることに特徴がある。

これは「ゃゅょ」は原則としてイ段かなの後にしか来ないこと、「っ」はカサタパ行の前にしか来ないことを利用している。
