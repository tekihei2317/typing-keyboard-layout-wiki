# 英月配列

30キーに収まっているコンパクトな月配列。

## 配列図

シフトは中指を使った前置シフト。

濁音は濁音シフト△を先に押してから、対応するキーを押す。

半濁音と小書き文字は、対応する仮名と同じ手の中指シフトキーを打ってから仮名キーを押す。

```text
単打

そきしてはをんいるー
すか★とたくう☆△；
ちこになさのり、。・

シフト

ゐおみえめへふせゆぬ
ねれらあわやまもよ；
ゑろけほむひつ　　・
```
