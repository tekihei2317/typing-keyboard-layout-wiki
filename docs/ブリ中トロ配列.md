# ブリ中トロ配列

TRONかな配列を中指シフト化したものをベースに、ハイブリッド月配列の拗音シフトを導入した配列。

記号を除き30キーの範囲に収めること、全てのモーラを2打以内で入力すること、同指連続を減らすことなどを設計方針に作られている。

清濁同値。ただ、右シフトと左シフトに規則性がなさそう。あと、は行は一部同値ではないものもある（例: ば、ぼ）。

ゃゅょ[T][Q][Z]を後置シフトキーとして利用する。イ段以外で拗音に接続するものはないので、ここに拗音を割り当てて2打鍵以内で入力できるようにする。

また、[DA][D;]も2打鍵の後置シフトで、これらは[ゅう][ょう]をホームポジションで入力するために使用する。

## メモ

とりあえず設計思想や差分を理解するために、古い方からどのような経過を辿っているか確認してみることにしよう。
