# TRON配列

TRONプロジェクトの中で作られた、キーボード・配列。キーボードは左右対称になっている。

シフト機構は同時打鍵を使用する。

## 配列図

文字キー範囲の30キーだけを抽出。

```text
単打

らるこはょきのくあれ
たとかてもをいうしん
まりにさなすつ、。っ

左シフト（親指、同時押し）

ひそ・ゃほぎげぐ　ゐ
ぬねゅよふ゛ぢゔじゑ
ぇぉせゆへずづ、。ゎ

右シフト（親指、同時押し）

びぞごばぼえけめむろ
だどがでぶおちーみや
ヵヶぜざべわぃぁ゜ぅ
```

ひとこと:

文字キーが30キーにおさまっている。濁音は反対の手のシフトと同時押しで打つ。濁音は、同じ位置にある単打または同手シフトのかなを濁音化するようになっている。
