## 拗音の排他的配置

拗音になるイ段のかながシフト面を含めて1つの場所に1つだけ配置されている。そのため、拗音が配置されているキー+ゃゅょとシフトを良略して2打鍵で拗音が打てる。

じ(aj) / り(kj) / び(dj) が重複しているが、"び"と"じ"の拗音は別のキーに配置されている。これは別の行段のルールで覚えることができる。

| --- | --- | --- | --- | --- | --- | --- | --- | --- | ------ | ------ |
|     |     |     |     |     |     | に  | ぢ  |     |        |        |

| Z   | ち  |     | し  |     |     | り  | B   |     |        | **ょ** |
|     |     |     | き  |     |     |     |     |     | **ゃ** |        |

## 行段の入力について

これは実質的にシフトキーが追加されているのと同じ。打鍵数は変わらないので効率面では良くも悪くもないかな。覚えやすさにメリットがある。

大事なのは「文字が割り振られる連節は、通常出現しないものでなければならない」こと。この条件を満たしているかどうかを一応確認してみよう。

|        |      |      |      |      |     |       |      |      |        |        |
| ------ | ---- | ---- | ---- | ---- | --- | ----- | ---- | ---- | ------ | ------ |
|        |      |      |      |      |     | li=ぃ |      |      |        |        |
|        |      | F=た | V=つ | W=を |     |       |      |      | lyu=ゅ |        |
| Z/A=？ | U=う | E=、 |      | L=が |     | O=お  | I=い | B=。 |        | lyo=ょ |
|        |      |      |      |      |     |       |      |      | lya=ゃ |        |

BZ行については、「。」と「？」をシフトと兼用している。P行については、これも前起シフト専用のキーになっている。

FVWL行については、イ段ではないので拗音との連節は存在しない。また、「い」「え」キーの代わりに「ぃ」「ぇ」キーを使うことで存在する連節を回避している。

| ----- | ---------- | ---------- | ---------- | ---------- | ---------- | ----------- | ---------- | ----------- | ---------- | ----------- |
|       | A=？       | I=い       | U=う       | E=、       | O=お       | lya=ゃ      | li=ぃ      | lyu=ゅ      | le=ぇ      | lyo=ょ      |
| B(。) | BA（。？） | BI（。い） | BU（。う） | BE（。、） | BO（。お） |             |            |             |            |             |
| Z(？) | ZA（？？） | ZI（？い） | ZU（？う） | ZE（？、） | ZO（？お） |             |            |             |            |             |
| P(ぱ) | PA（ぱ？） | PI（ぱい） | PU（ぱう） | PE（ぱ、） | PO（ぱお） | PYA（ぱや） |            | PYU（ぱゅ） |            | PYO（ぱょ） |
| F(た) |            |            |            |            |            | FA（たゃ）  | FI（たぃ） |             | FE（たぇ） | FO（たょ）  |
| V(つ) |            |            |            |            |            | VA（つゃ）  | VI（つぃ） |             | VE（つぇ） | VO（つょ）  |
| W(を) |            |            |            |            |            |             | WI（をぃ） |             | WE（をぇ） | WO（をょ）  |
| L(が) |            |            |            |            |            | LA（がゃ）  | LI（がぃ） | LU（がぅ）  | LE（がぇ） | LO（がょ）  |

これで比較的メジャーな外来音で2打鍵で打てないのは「とぅ」「どぅ」あたりだろうか。ほとんど使わないけど。

W + ゃを「わ」にするのか「うぁ」にするのか、W+ょを「を」にするのか「うぉ」にするのかなどは気になる。ローマ字に従うなら前者。

## メモ

ハイブリッド月配列で「キー共有」と呼ばれているのは、2打目で直前に打った文字をキャンセルすること。

句読点とシフトを兼ねているところで、これによって打鍵効率をあげている。

他にも、「ゃゅょ」もキー共有機能がある。

拗音になるイ段のキーを排他的配置して、2打で拗音を打てるようにしている。

作成者によって3年間実用された実績がある。
