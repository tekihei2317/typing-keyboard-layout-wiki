# 花配列

中指前置シフトの先駆けとなった配列。月配列と比べると上段の使用率が高い特徴がある。

## 配列図

```
単打
ょてとこは っくうるらー
すか☆きた んれ☆゛ろり
さしなのに しつ。、めえ

☆前置
ひけぇほへ ゃまそもぃ「
ぁよゅや゜ よふちむぉ」
ぅせあわゆ ねみをおぬ
```
