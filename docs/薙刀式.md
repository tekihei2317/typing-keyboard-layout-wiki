# 薙刀式

監督・脚本家の大岡俊彦氏によって作られた、長文を楽に速く書くための配列。

「ある」「ない」「する」の日本語の骨格となるパーツが、右手ホームポジションにアルペジオで打てるように配置されている。

シフト方式はセンターシフトを使った連続シフト。また、濁音、拗音、外来音などは構成する文字キー＋αの同時押しで入力することができ、記憶負担が小さい特徴がある。

## タイピングゲームでの実績

大岡さんはタイプウェルSSとのことだが、打鍵動画を見る限り基本常用語XF~XD程度相当の実力はあると思われる。

競技レベルでの高速タイピングではシフト化けが課題となる。タイプウェルでは日本語の文章では通常使わないスペースを多用するため、SandSとの相性が少し悪い。

他にも、拗音シフト→くるへ、濁音半濁音シフト→かあこな、がシフトキーを兼用しているため、これらの前にシフト化する文字がくる場合はキーを離すことを意識する必要がある。例えば、たか→だ、とる→にょ、に化ける可能性がある。
