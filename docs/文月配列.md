# 文月配列

日本語の文章を楽しく、効率的に書くことを目的に設計された、月系の計算配列。

設計には3000万文字の日本語データセットが使われている。評価指標は、打鍵時間、悪運指率、指の負担など。

「が」「で」などの頻出濁音が単打で、それ以外は清濁同置になっており、打ちやすさと覚えやすさのバランスが取られている。

また、濁音や小書きが排他的配置されているので、全てのかなを2打鍵以内で打つことができる。

## 配列図

薬指逆手シフトで濁音化、薬指同手シフトで半濁音化する（濁音の排他的配置）。

中指同手シフトで小書き化する（小書きの排他的配置）。

