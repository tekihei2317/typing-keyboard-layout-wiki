# 月配列2-263式

新JIS配列を前置シフトにするとどうなるか？というアイデアで作られた配列。2ch生まれ。

出現率の高いかなは1打で、それ以外のかなは2打で打つ。シフトはD/Kキーをシフトに使う中指前置シフト。打ちたいかなと反対のシフトキーを押すクロスシフトが基本。

2-263式が月配列の代表例で、その派生が多数ある。

## キー配列

通常面

そこしてょつんいのりち<br>
はか☆とたくう★゛きれ<br>
すけになさっる、。゜・<br>

シフト面

ぁひほふめぬえみやぇ「<br>
ぃをらあよまおもわゆ」<br>
ぅへせゅゃむろねーぉ　<br>

Zタイパーが複数名あり、JISかな以外の配列では最も実績がある配列だと言える。

カタカナ語と慣ことの記録は1~2ランク下がっている。カタカナ語については、頻出の長音（ー）や拗音（ゃゅょ）・外来音（ぁぃぅぇぉ）に2打鍵必要なため難易度が上がっていると考えられる。

↑ょは1打鍵。
