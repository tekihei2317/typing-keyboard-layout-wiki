# 幸花配列

幸花（ゆきか）配列。月配列4-698式、⚪︎（まる）配列配列とも呼ばれる。

計算配列の先駆けで、遺伝的アルゴリズムを使って配置を決めている。

## 配列図

```text
単打
こなとけろ　むるいのれめ
たて☆かは　っん☆゛しー
すょせきさ　くう、。゜

☆前置
ぁねやほゆ　ぬわあにぇ「
ぃおらもふ　ゃつまちえ」
ぅゅそよへ　みりひをぉ
```
