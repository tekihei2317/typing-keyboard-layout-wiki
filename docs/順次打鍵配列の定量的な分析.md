# 順次打鍵配列の定量的な分析

英語新配列を参考に、新配列も定量的に分析してみたい。準備して進めていく。

## 準備

### 評価指標を用意する

- 2-gram（SFB、SFS、Scissors）
- 3-gram（Alt、Rolls、Redirect）
- あとはヒートマップとか

### 評価指標の計算方法を理解する

- 評価指標の計算方法を理解する

### 順次打鍵の配列を用意する

ローマ字とかなの両方を比べたい。

かなはキー範囲が広いと色々計算がややこしくなりそうなのでJISかなは一旦保留しよう。

- ローマ字

  - 大西配列
  - M式
- 拡張ローマ字入力
  - けいならべ
  - SKY
  - AZIK
- かな
