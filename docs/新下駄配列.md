# 新下駄配列

同時押しを使用するかな配列。清濁別置で、拗音や外来音も1アクションで入力できる。

かなの配置は、日本語の頻度の多い2連節が打ちやすいように考えられている。

## 配列図

```txt
単打

ーには、ち ぐばこがひげ
のとかんっ くういしなBS
すまきるつ てたで。ぶ

中指シフト（反対の手の中指の同時押し）

ﾌｧごふﾌｨﾌｪ ｳｨぱよみｳｪｳｫ
ほじれもゆ へあ■■え
づぞぼむﾌｫ せねべぷヴ

薬指シフト（反対の手の薬指の同時押し）

ぢめけﾃｨﾃｨ ｼｪぺどやｼｪ□
をさおりず びら■■そ
ぜざぎろぬ わだぴぽﾁｪ

拗音シフト（Iと同時押し。左側のみ。）

ひゅしゅしょきゅちゅ
ひょ■■■■きょちょ
ひゃ■■しゃきゃちゃ

拗音シフト（Oと同時押し。左側のみ。）

りゅじゅじょぎゅにゅ
りょ■■■■ぎょにょ
りゃ■■じゃぎゃにゃ
```

拗音シフトの■の箇所は文字が配置されていない。

中指シフトと薬指シフトの■の箇所は、別の図では「へあれおね」「びらじさそ」になっているが、他の打ち方があるので省略していると思われる。
